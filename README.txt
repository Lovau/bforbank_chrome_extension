Permets à un client de Bforbank, quand il est connecté à son compte et qu'il est sur une page de relevés de compte (compte courant ou livret), de télécharger la liste des relevés au format CSV.

Aucune information n'est stockée par l'extension, aucune donnée n'est transmise.

Le fonctionnement de cette extension est simple : 
- Vérifier l'URL pour détecter une page de relevés de compte bforbank (ie : https://client.bforbank.com/espace-client/consultation/operations/* )
- Formatter en CSV le tableau de relevés affiché
